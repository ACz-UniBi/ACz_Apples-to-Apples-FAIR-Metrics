14-http-describedby-citeas-linkset-json-txt-conneg

This landing page has four HTTP headers:

  • describedby to index.ttl (text/turtle, type attribute not declared)
  • cite-as to w3id
  • linkset to linkset with content negotation for application/linkset+json that links to test-apple-data.csv and the above
  • linkset to linkset with content negotation for application/linkset alternative to the above

There is NO Link rel=alternate between JSON and TXT as they have the same public URI. The Web Server should give a specific Content-Location header of which resource was resolved.

Bonus: Content types application/json and text/plain also content-negotiate with lower priority.

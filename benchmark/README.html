Persistent Identifiers

There is a corresponding Persistent Identifier (PID) for each test, given as https://w3id.org/a2a-fair-metrics/ suffixed with the test name listed above, e.g. https://w3id.org/a2a-fair-metrics/03-http-citeas-only/ but note that not all tests neccessarily have the corresponding cite-as header.

Note that some of the tests (e.g. 04-http-described-iri) deliberately refer to resources using IRIs to the alternative hostname з11.ею (aka xn--11-slc.xn--e1a4c) instead of s11.no as pointed to by the PID redirects. It is also possible to force access to the tests using IPv6 only with the hostanmeI ipv6.s11.no.
